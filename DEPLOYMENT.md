# Deploy Project Portal to GitHub Pages

1. **Push Changes**:
   - Open **GitHub Desktop**.
   - Commit the new `index.html` file (Message: "Convert to static site").
   - Push to `origin`.

2. **Enable GitHub Pages**:
   - Go to your repository on GitHub.com.
   - Click **Settings** (top right tab).
   - Sidebar: **Pages**.
   - **Build and deployment** > Source: **Deploy from a branch**.
   - **Branch**: Select `main` (or `master`) and folder `/ (root)`.
   - Click **Save**.

3. **Done!**
   - Wait about 1 minute.
   - Refresh the page, you will see your live URL (e.g. `https://yourname.github.io/project-portal/`).
   - It will load INSTANTLY.
